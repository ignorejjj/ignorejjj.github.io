金佳杰，中国人民大学高瓴人工智能学院2025级博士生，导师为窦志成教授，主要研究方向为智能信息检索、检索增强生成和大语言模型智能体。以第一/共同第一作者身份在 ACL、SIGIR、The Web Conference、NeurIPS 等国际顶级会议发表7篇论文，其代表工作包括 FlashRAG、FinSight、WebThinker、Search-o1 等, 谷歌学术引用1900+，受到国内外研究者和开源社区的广泛关注。在学术研究以外，其主导构建了多个开源项目，提出的FlashRAG框架曾进入 GitHub 趋势榜前三名，项目获得 3.5k+ GitHub Stars，PyPI 下载量超过 8 万次。设计的FinSight金融研报生成系统在2025年AFAC 金融智能创新大赛中获得第一名（1/1289）。参与的 WebThinker 和 Search-o1 项目也分别获得 1k+ GitHub Stars，推动了大语言模型深度研究与搜索增强推理方向的发展。曾在阿里通义实验室、微软亚洲研究院开展科研实习，获江淮蔚来特别奖学金、中国科大一等学业奖学金等。
