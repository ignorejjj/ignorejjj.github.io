金佳杰，中国人民大学高瓴人工智能学院2025级博士生，导师为窦志成教授，主要研究方向为智能信息检索、检索增强生成和大语言模型智能体。以第一/共同第一作者身份在 ACL、SIGIR、The Web Conference、NeurIPS 等国际顶级会议发表7篇论文，其代表工作包括 FlashRAG、FinSight、WebThinker、Search-o1 等, 谷歌学术引用1900+，受到国内外研究者和开源社区的广泛关注。在学术研究以外，其主导构建了多个开源项目，提出的FlashRAG框架曾进入 GitHub 趋势榜前三名，项目获得 3.5k+ GitHub Stars，PyPI 下载量超过 8 万次。设计的FinSight金融研报生成系统在2025年AFAC 金融智能创新大赛中获得第一名（1/1289）。参与的 WebThinker 和 Search-o1 项目也分别获得 1k+ GitHub Stars，推动了大语言模型深度研究与搜索增强推理方向的发展。曾在阿里通义实验室、微软亚洲研究院开展科研实习，获江淮蔚来特别奖学金、中国科大一等学业奖学金等。

我的代表性工作沿三条主线展开：

（1）RAG 基础设施与方法：主导开发模块化 RAG 研究工具包 FlashRAG（TheWebConf 2025 Oral，GitHub 3.5k+ star，全球下载量 7 万余次，论文引用 200+），覆盖 30+ 先进 RAG 算法；并提出长文档精炼框架 LongRefiner（ACL 2025 Oral）、知识桥接方法 BIDER（ACL 2024）等系统性方法，奠定大模型高效利用外部知识的基础。

（2）深度搜索与垂直领域智能体：主导开发 AI 深度搜索引擎 WebThinker（NeurIPS 2025，共同一作，GitHub 1.4k+ star），赋予推理大模型自主网页搜索与深度研究能力；提出层次化多智能体推理框架 HiRA（SIGIR 2026）与搜索增强推理范式 Search-o1（EMNLP 2025 Oral，1k+ star），构建从规划到执行解耦的深度搜索体系；并落地金融深度研究系统 FinSight（ACL 2026 Oral, Best Paper Candidate），获 AFAC 金融智能创新大赛全国一等奖（1/1289），相关技术已申请国家发明专利。

（3）可信与长程自主研究智能体：面向自主智能体的两大核心挑战——可信性与长程性，提出可审计推理框架 VeriGraph（投稿 NeurIPS 2026），将智能体推理轨迹建模为可验证的证据图，实现数值可复现、结论可追溯；并在微软亚洲研究院（MSRA）实习期间主导自主研究框架 Arbor，以"假设树精炼"将零散试验沉淀为持久研究状态，实现长程自主研究与自我优化。
